# 🚀 Feature Engineering & Model Performance Comparison

## 📌 Project Overview

This project demonstrates feature engineering, feature scaling, model optimization, and performance comparison using the California Housing Dataset.

## 🎯 Objectives

- Load and preprocess the California Housing Dataset
- Apply Feature Scaling using StandardScaler
- Train multiple regression models
- Compare model performance using RMSE and R² Score
- Identify the best-performing model

## 📂 Dataset

- California Housing Dataset
- Source: Scikit-learn
- Total Records: 20,640
- Features: 8
- Target: House Price

## 🤖 Models Used

- Linear Regression
- Ridge Regression
- Decision Tree Regressor

## 📊 Evaluation Metrics

- Root Mean Squared Error (RMSE)
- R² Score

- Python

- Scikit-learn
- Jupyter Notebook
